# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities to the GitHub repository's Security tab or email carlosrocha@users.noreply.github.com.

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Scope

This project is an SBOM generator for AI models. Vulnerabilities include:
- Integrity bypass in manifest verification
- Code execution via malformed model metadata
- Supply chain attacks via compromised dependencies

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅        |
